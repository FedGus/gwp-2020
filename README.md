## Установка проекта

```
npm install
```

### Компиляция и хот-релоад для разработки

```
npm run serve
```

### Компиляция и минификация для продкшена

```
npm run build
```

### Запустить unit-тесты

```
npm run test:unit
```

### Запустить end-to-end тесты

```
npm run test:e2e
```

### Исправление и проверка красоты кода

```
npm run lint
```
